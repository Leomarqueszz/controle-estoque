# Controle de Estoque

## Sobre o projeto

Esse projeto foi desenvolvido com o objetivo de praticar Python e banco de dados em uma situação simples do dia a dia de uma empresa.

A ideia foi criar um sistema de controle de estoque onde é possível cadastrar produtos, consultar os itens cadastrados e atualizar as quantidades disponíveis.

## Tecnologias utilizadas

Python

SQLite

## Funcionalidades

Cadastro de produtos
Consulta de produtos
Atualização de estoque

## Como pensei no projeto

Quando comecei a desenvolver, pensei em um problema que praticamente toda empresa possui: controlar seus produtos.

A partir disso, criei uma tabela para armazenar as informações básicas de cada produto, como nome, quantidade e preço. Depois desenvolvi as funções para cadastrar novos produtos, visualizar os produtos já cadastrados e atualizar o estoque quando necessário.

O principal objetivo desse projeto foi praticar a integração entre Python e banco de dados, além de desenvolver minha lógica de programação e organização de dados.
